# 🧠 Descripción del Proyecto

**Adventure Game** es un juego de aventura interactivo desarrollado con **React + TypeScript**, donde el jugador explora escenarios, toma decisiones y avanza a través de una historia dinámica.

El objetivo es ofrecer una experiencia ligera, modular y escalable basada en componentes reutilizables.

---

# 🕹️ Características

✔ Sistema de decisiones (story branching)
✔ Escenarios dinámicos
✔ UI basada en componentes
✔ Estados del juego controlados con React Hooks
✔ Tipado estricto con TypeScript
✔ Diseño responsivo adaptable a móvil y desktop

---

# 🧩 Tecnologías Utilizadas

## ⚛️ Frontend

* React

* React Hooks (useState, useEffect)

# 📁 Estructura del Proyecto

```text id="proj1"
src/
│
├── components/      # UI reutilizable
├── scenes/          # Escenarios del juego
├── data/            # Historia y decisiones
├── hooks/           # Lógica personalizada
├── context/         # Estado global del juego
├── assets/          # Imágenes y sonidos
├── App.tsx          # Entrada principal
└── main.tsx
```

---

# 🎯 Cómo funciona el juego

1. El jugador inicia la historia
2. Se muestran decisiones en pantalla
3. Cada elección cambia el rumbo del juego
4. El estado se actualiza dinámicamente
5. Se desbloquean nuevos escenarios

# 🚀 Instalación

```bash id="inst1"
# Clonar repositorio
git clone https://github.com/tu-usuario/adventure-game

# Entrar al proyecto
cd adventure-game

# Instalar dependencias
npm install

# Ejecutar proyecto
npm run dev
```
